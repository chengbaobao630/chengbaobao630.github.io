C++ 学习1

之前已知用的是java做后端的web开发，最近苦逼与要自考 学习了C++
c++ 与java 都属于是静态类型语言->其意思是 数据的类型在编译期就已经决定好了，但是他们也有共同的泛型特性，在运行期间决定类型
c++ 比较c是对其的一种扩展

语法

定义变量

大部分还是与java 类似，只是有少许的差别如：Student s;在java中定义了一个 空指针的引用，而在c中是直接初始化了一个对象，他的生命周期由C++来管理
二 通过new Student()初始化的对象需要手动的delete来销毁。

定义方法

这个也是大部分类似的，只是会设计传参，实参，引用和指针，还有返回值上的区别之后在再说。

结构

c++ 中的作用域指定 是真对块，而且默认是private
java 中是对单条，默认是包内可见
c++中类中不允许对字段进行初始化。

引用别的文件

java中大家都知道使用import来引入需要导入的包，它主要是用来定位jvm中类加载器中的唯一的类（就算是同一个类，不同的类加载器来加载他在jvm中也是两个不一样的类）
c++中有两写法

第一种是引入用户自定义的头文件
第二种是引入一些系统级别的（不是很了解，就和jvm内置的差不多一个意思吧）文件
这些引入好像最终都会被加载进文件中，也不是很了解。。。
